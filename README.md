# Fogbound Mafia

브라우저에서 실행되는 마피아 게임과, 이를 감싼 안드로이드 WebView 앱 프로젝트가 함께 들어 있습니다.

## 웹 버전

`D:\project\host\index.html` 파일을 브라우저로 열면 됩니다.

## 안드로이드 앱

- 프로젝트 루트: `D:\project\host`
- 앱 모듈: `D:\project\host\app`
- WebView가 `app/src/main/assets/web/index.html`을 로드합니다.

## 현재 기능

- 플레이어 수 선택: 3명부터 8명까지
- 플레이어 이름 입력 없이 `플레이어 1`, `플레이어 2` 형식으로 진행
- 인원 수에 따라 의사와 경찰 최대치 자동 제한
- 직업 수 선택: 마피아, 의사, 경찰
- 남는 인원은 자동으로 시민 배정
- 밤 행동: 마피아 제거, 의사 보호, 경찰 조사
- 낮 토론 타이머 자동 시작 및 수동 리셋
- 모바일 한 화면 중심 레이아웃
- 사건 기록 드로어로 내레이션 확인
